## ⛳ Featured Projects

### 🌦️ WeatherCaddy
**Mobile app for smarter golf decisions**

WeatherCaddy helps golfers decide *where* and *when* to play by analyzing real-time weather conditions and translating them into a simple, actionable score.

**Key Features**
- Built with **React Native**
- Save and compare favorite golf locations
- Location-based **1–100 playability rating**
- Factors in:
  - Temperature
  - Wind speed
  - Rain probability
  - Air density
  - Additional environmental conditions
- Designed for quick, at-a-glance decision making

### 🏌️ TrackmanCaddy
**Automation assistant for TrackMan users**

TrackmanCaddy is a background utility designed to speed up and streamline workflows when using TrackMan software. It reduces repetitive clicking and manual input so you can focus on practice, not menus.

**Key Features**
- Built with **Python** and **shell scripting**
- Runs silently in the background
- Screen scanning for automated GUI interactions
- Keyboard listeners to trigger common actions
